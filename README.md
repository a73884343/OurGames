# لعبتنا معاً 💜

موقع ألعاب للزوجين — multiplayer حقيقي بـ Socket.io

## الميزات
- 🧠 اختبار ثقافي تنافسي (نفس الوقت)
- 🗣️ وصف وخمّن (تعاوني)
- 🧩 تركيب الصور (تعاوني)
- 💬 شات خاص بين اللاعبين
- ✨ تحدي اليوم وسؤال اليوم
- ❤️ إرسال المزاج

## الرفع على Railway (أسهل طريقة)

### الخطوات:
1. ارفع الملفات على GitHub repo جديد
2. افتح [railway.app](https://railway.app) وسجل دخول
3. New Project → Deploy from GitHub repo
4. اختر الـ repo
5. Railway يكتشف Node.js تلقائياً ويشتغل

بعد الرفع تحصل على رابط مثل:  
`https://couple-game-production.up.railway.app`

## التشغيل محلياً

```bash
npm install
npm start
```

ثم افتح `http://localhost:3000`

## كيف تلعب
1. الأول يفتح الموقع ويكتب اسمه → ينشئ غرفة
2. يكوبي الكود ويرسله لحبيبته
3. هي تفتح الموقع وتكتب اسمها → تنضم بالكود
4. استمتعوا! 🎮
